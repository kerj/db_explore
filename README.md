# What

An Express app that supports server-side rendered react components. 

## Installation


```bash
yarn

yarn webpack
```
In a separate terminal:
```bash
yarn dev
```

Navigate to `/ssr`

is setup to be ran using the demo database generated from 
[this repo](https://github.com/kerj/sql_mock_data_gen)
